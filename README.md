# Lumberjack-ESX---Hungary
Ez a script teljes ESX integrációval érkezik, kifejezetten a magyar közösséghez igazítva.  Fő funkciók:  Faanyag gyűjtés és feldolgozás  Eladás és fizetés ESX rendszerben  Egyedi magyarországi járművek és munkahelyi ikonok  Egyszerű telepítés: ZIP fájlban, részletes útmutatóval
